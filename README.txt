Task 1: Citesc valoarea din fiser pentru NC, iar pentru points utilizez
o functie de transformare a unei structuri intr-o matrice.

Task 2: Se initializeaza valorile pentru maxime si minime, apoi se creeaza centroizii.
Se gaseste distanta minima pentru fiecare punct si seatribuie in functie de distanta
fata de centroizi. Apoi, se recalculeaza pozitiile centroizilor ca fiind centrele
de masa ale punctelor, repetandu-se pasii pana cand pozitiile centroizilor
nu se mai modifica.

Task 3: Se salveaza marimile matricelor si se initializeaza vectorul de culori.
Se parcurg matricele si se determina distanta minima a fiecarui punct.
La final, se reprezinta punctele colorate in functie de centroidul caruia apartin.

Task 4: Se salveaza marimile matricelor. Se parcurg matricele si se determina distanta
minima intre puncte, apoi se calculeaza costul in functie de fiecare distanta 
minima obtinuta.

Task 5: Se initializeaza o matrice cu valorile din fisier, dupa schimbarea
structurii in matrice. Apoi, se parcurg 10 clustere si pentru fiecare se calculeaza
un vector de costuri. La final, se reprezinta grafic costul total in funtie de 
numarul de clustere, cu ajutorul functiei plot.
